# ⚽ KushFC Automation
Automated football results generator for KushFC brand.

## 🚀 How it works
This project uses **Python** and **GitHub Actions** to:
1. Fetch latest football results from **Football-data.org**.
2. Process the data and overlay it on a custom branded background.
3. Generate ready-to-post images for social media.

## 🛠️ Setup
- Background image: `background.jpg`
- Main script: `main.py`
- Automation: `.github/workflows/main.yml`
